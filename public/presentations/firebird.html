What is Firebird?

Firebird is the project name for the latest generation of consumer-facing Bazaarvoice applications.

Highlights

  • 100% Javascript. Server and Client.
  • We only ship a static clientside file.
  • Uses the SAME Dev API as our clients.
  • Client-side rendered.
  • Reduce the # of requests to the server.
  • Minimize the size of the entire application.

Cool Things about Firebird

Magic Firebuild.

We custom build a javascript app for each client based on the their bhive configuration. That's cool for number of reasons.

  • We can build out the code that we don't need
  • The whole app is a static Javascript file, which lives in Amazon S3. That means we don't need to have a dedicated end-user facing server up running all the time. Big win on the reliability.
  • All the dynamic content comes from API

Modular?

3 Concepts:

  • Components
  • Features
  • Outlets

What is a Component?

We consider a component as a standalone UI widget, which can function properly by itself and interacts with other components as well.


      reviewContentList1 : {
        componentId : 'reviewContentList1',
        container : 'BVRRContainer',
        subjectType : 'Products',
        contentType : 'Reviews',
        type : 'contentList',
        autoload : true,
        features : {
          ...
        },
        outlets : {
          ...
        }
      }

What is a Feature?

Feature is a smaller UI element, which usually belongs to a larger feature/component. A feature shall be able to turn on and off easily.


reviewContentList1 : {
  ...
  features : {
    self : ['contentItemCollection', 'pagination', 'ugcCount'],
    contentItem : ['has:stars', 'avatar', 'feedback'],
    pagination : ['ugcCount']
  },
  ...
}

What is an Outlet?

Outlet is the bridge that connects different components.


reviewContentList1 : {
  ...
  outlets : {
    showsubmission : [{
      component :'reviewSubmission1',
      scope : 'rr',
      action : 'submit_review'
    }],
    showmediaviewer: [{
      component: 'mediaViewer1',
      event: 'open'
    }],
    showfullprofile : [{
      component : "fullProfilePopup1",
      event: "launchprofile"
    }]
  }.
  ...
}

How to Wire Events between Components?


reviewSummary1 : {
  ...
  outlets : {
    showreviews : [{
      component : 'reviewContentList1',
      event : 'scrolltocontent'
    }],
    ...
  }.
  ...
}
            

reviewSummary1 : {
  ...
  outlets : {
    showreviews : [{
      component : 'reviewContentList2',
      event : 'scrolltocontent'
    }],
    ...
  }.
  ...
}

How to turn on/off a feature?


      reviewSummary1 : {
        ...
        features : {
          self : []
        },
        ...
      }
            

      reviewSummary1 : {
        ...
        features : {
          self : ['histogram', 'has:stars']
        },
        ...
      }
